# Asset Notes

## Shirt Artwork
Files:
- `pixel-shirts.svg` (base)
- `pixel-shirts-sm.svg` (160×100)
- `pixel-shirts-md.svg` (320×200)
- `pixel-shirts-lg.svg` (640×400)

These are vector assets and can scale cleanly. The size-specific files are convenience copies for layout testing.

If you want to tweak the silhouette, edit `pixel-shirts.svg` and copy changes to the size variants.
